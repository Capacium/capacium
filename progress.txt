## 2026-04-23: Phase 2 — Full Implementation (5 Workstreams)

### What was done
- **WS-BUNDLE**: Bundle support for kind:bundle — recursive sub-cap installation, removal with reference counting, bundle fingerprinting, list/search --kind filtering, bundle member tracking via `bundle_members` table
- **WS-ADAPTERS**: Claude Code (`adapters/claude_code.py`) + Gemini CLI (`adapters/gemini_cli.py`) framework adapters with adapter auto-selection based on manifest `frameworks:` field
