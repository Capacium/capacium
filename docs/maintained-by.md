# Maintained By

Capacium is maintained by [**fusionaize.com**](https://github.com/fusionAIze).

## About fusionAIze

fusionAIze builds capability infrastructure for the AI agent ecosystem — from packaging and discovery (Capacium) to workflow orchestration (SkillWeave).

## Open Source

Capacium is open-source software. The core CLI, Exchange API, crawler, and all adapter code are publicly available under the project's license.
